# 台灣第一家烏龍嗨 · The Oolong High

> 日常但不一般的燒酎下午酒場
> 帶著 90 年代喫茶店的氣味，現在喝得到。

品牌介紹網站。包含品牌定位、酒單設計系統、燒酎宇宙地圖等內容。

## 📄 頁面

| 頁面 | URL | 用途 |
|---|---|---|
| **品牌中心** | `/` (index.html) | 主入口 — 品牌介紹、4 張力、5 圈系統概覽 |
| **燒酎宇宙地圖** | `/燒酎宇宙地圖.html` | 10 張互動式風味地圖（願景視角）|
| **5 圈系統詳解** | `/5圈系統詳解.html` | 酒單設計系統的完整教學 |

## 🛠 Tech Stack

- 純 HTML + CSS + 原生 JS（無框架）
- 自製 SVG 視覺化（矩陣、季節輪、風味光譜、文化光譜）
- Plotly.js（部分圖表）
- Google Fonts（Noto Serif TC、Noto Sans TC、Inter）

## 🎨 Design

- 暖米色系（warm paper palette）
- 茶褐 / 麥色 / 焙茶橙 / 墨色
- Noto Serif TC + Sans TC + Inter 三軌字體
- 手繪感 SVG 取代工業感 chart 元件

## 🚀 Deploy

Static site，可直接部署到 Vercel / Netlify / Cloudflare Pages 任一平台。
