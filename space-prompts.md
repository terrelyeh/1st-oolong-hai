# 台灣第一家烏龍嗨 · 空間示意圖 Prompts

> 給 AI 生圖工具用的提示詞集（Midjourney / Flux / Nano Banana / DALL-E 通用）。
> 重點：**空間「氣味」的編輯影像**，不是室內設計圖、不是 3D render。

---

## 品牌空間 DNA（所有 prompt 共通的視覺語彙）

| 維度 | 是 | 不是 |
|---|---|---|
| **色溫** | 3000K 鎢絲燈、暖琥珀、午後斜陽 | 冷白 LED、霓虹、舞台聚光 |
| **材質** | 暖橡木、米色亞麻、和紙、銅、磨砂玻璃、皮革 | 黑色大理石、鏡面、不鏽鋼、混凝土 |
| **配色** | 米白 / 茶褐 / 焙茶橙 / 麥色 / 墨色 | 高飽和、純黑純白、撞色 |
| **參考時代** | 90 年代日式喫茶店 + 下町居酒屋 + 老派唱片行 | speakeasy、hotel bar、IG 網美店 |
| **狀態** | 有歲月、被使用過、被照顧 | 全新、刻意做舊、樣品屋 |
| **情緒** | 慢、安靜、午後、可獨處可結伴 | 派對、熱鬧、打卡 |

---

## 通用樣式後綴（每個 prompt 結尾接這段）

```
Editorial photography, 35mm film grain, warm tungsten 3000K lighting,
cream / amber / tea-brown / hojicha palette, lived-in not staged,
Kinfolk × Apartamento × Hirokazu Kore-eda sensibility,
no chrome, no marble, no neon, no LED. Aspect ratio 3:2.
```

## 通用負面詞（Negative prompt）

```
black marble, polished chrome, mirror wall, industrial concrete,
cold blue LED, fluorescent ceiling, neon sign, speakeasy hidden door,
hotel lobby bar, all-white minimal, IKEA showroom, fake plants,
people posing for camera, hyperreal 3D render, architectural visualization,
HDR, oversaturated.
```

# 🎵 Style Track B · 90 年代 City Pop / 數位懷舊風（底片攝影路線）

> 用途：跟前面 11 個鏡頭（**Style A · 編輯攝影**）並列。
> **A 與 B 都是真實照片**，差別在於：
>   - **A** = 當代編輯攝影、自然光、靜謐午後、紀錄感（Kinfolk × Kore-eda）
>   - **B** = 1990 年代日本 / 香港 / 台北的夜晚街拍底片、霓虹、有點髒、有點熱、有點失焦（Wong Kar-wai × TO:KY:OO × Cinestill）
> 你既有的 poster #7（浮世繪 × glitch）是 B 線的「概念海報延伸」，但**底層美學就是這個 1992 年的霓虹底片感**。

---

## City Pop（底片攝影版）DNA

| 維度 | 是 | 不是 |
|---|---|---|
| **媒材** | 35mm / 中片幅底片照片、自然光、手持 | 插畫、油畫、3D render、向量 |
| **時代感** | 1985–1995、霓虹、卡帶、CRT、撥盤電話、磨砂玻璃 | 60s 復古、2010 後街拍、現代 minimal |
| **色彩** | 桃紅 / 青藍 / 鈉燈橘 / 鎢絲暖黃 / 深紫午夜 | 高飽和 Vaporwave、單色、現代 HDR |
| **質感** | 底片顆粒、霓虹邊緣色散、輕微 light leak、燈絲光暈 | 4K 銳利、智慧型手機、攝影棚閃燈 |
| **氛圍** | 午夜剛下雨、便利商店燈光、最後一班電車、Plastic Love | 派對 hype、IG 打卡感、新聞紀錄感 |
| **物件元素** | 卡帶、Pioneer 唱機、CRT 電視、霓虹招牌、撥盤電話、磨砂玻璃 | 智慧型手機、平板、現代 UI、LED 螢幕 |

## 視覺參照（攝影為主）

**電影 / 攝影：**
- Wong Kar-wai 王家衛 `Fallen Angels` (1995)、`Chungking Express` (1994) — 霓虹街拍、慢快門人影
- Liam Wong `TO:KY:OO` 攝影集 — 雖是 2010s 但完美復刻 1990s 東京霓虹底片感
- Daido Moriyama 森山大道 — 高反差、底片粒、街拍
- Greg Girard `Hong Kong: Portraits of Power`、`KOWLOON Walled City` — 1980s 港式霓虹
- Nobuyoshi Araki 荒木經惟（早期街拍部分）— 80s 東京色感

**底片配方（請寫進 prompt）：**
- `Cinestill 800T` — 鎢絲色溫修正、霓虹邊緣紅暈光暈，**夜景神片**
- `Kodak Portra 400` — 暖色系、人像、夕陽
- `Fuji Velvia 50` — 飽和、適合戶外白天
- `Kodak Gold 200` — 黃綠偏色、復古日常

**相機：**
- Nikon FM2、Pentax 67、Contax T2、Yashica T4
- 都是 90s 真的會用的隨身機 / 中片幅

**情境參考：**
- 1990 年代 Suntory 威士忌、Asahi 啤酒、寶酒造（燒酎）的雜誌平面廣告
- Tokyo / 新宿歌舞伎町 / 渋谷 1992 的紀錄街拍
- 台北 1990 年代林森北路、條通、五分埔

**City Pop 唱片封面當「氣味參照」（不是視覺直接抄）：**
- 竹内まりや `Plastic Love (1984)`、杏里 `Timely!! (1983)`、山下達郎 `For You (1982)`
- 用來校準色感和氛圍，不是要畫成插畫

## A vs B · 怎麼分

| 用途 | 用 Style A（編輯攝影） | 用 Style B（City Pop） |
|---|---|---|
| 品牌書內頁主視覺 | ✅ | （插頁可用） |
| 官網 hero / 章節分隔 | ✅ | ✅（看頁面氣味） |
| IG 主版 feed | ✅ | ✅ |
| 限定活動海報 / flyer | （太靜了） | ✅✅ 最對 |
| T-shirt / 周邊 / 帆布袋 | （難印） | ✅✅ |
| 季節限定主題 | ✅ | ✅（夏天最對） |
| 給投資人看 | ✅✅ | （可能太復古） |
| 給夜貓子客人 | （白天感） | ✅✅ |

**一個簡單原則：**
- A 講「我們是什麼樣的店」
- B 講「我們屬於什麼樣的時代 / 文化」

兩條軸線都在的話，品牌會立體得多。

## 用法建議

### 工具差異

| 工具 | 建議 |
|---|---|
| **Midjourney v6+** | 直接貼整段 prompt + `--ar 3:2 --style raw`，先 V4 一輪挑構圖再 upscale |
| **Flux / Nano Banana** | 整段貼，可以再加「`composition: editorial photo, rule of thirds`」 |
| **DALL-E 3 / Sora** | 把 prompt 縮短到 200 字內，移除過多形容詞，留下關鍵物件 + 風格 |
| **Stable Diffusion** | 用通用負面詞、加 LoRA：`Kodak Portra 400` / `editorial film` |

### 套組建議

如果要產一整套品牌視覺，建議：
1. 先生 **01 + 04**（最具代表性的兩張，風格基準）
2. 用同一個 seed / style ref 接著生 **02 / 03 / 05 / 06**（保持氣味一致）
3. **不要每張都換風格描述**——同樣的 6 個鏡頭、同樣的後綴，整套才會像一家店

### 跟現有 7 張 poster 的關係

- Posters（#1–#7）= **品牌符號 / 概念海報**（拼貼、印刷感、抽象）
- 空間示意圖 = **品牌世界觀 / 場景現實**（編輯攝影、具象、可被想像走進去）

兩者一起，等於有了「概念面」+「現實面」的完整品牌視覺資產。

## 想試其他鏡頭？

延伸方向（還沒寫的）：
- **下班時段的窗外**（玻璃內外光線對撞、可放在「下午酒場」概念上）
- **菜單被翻開的瞬間**（連結 #6 海報但是現實版）
- **雨天的店面 / 內部**（換天氣換氣味、不一樣的午後）
- **季節限定 / 4 季 4 張**（春櫻 / 夏抹茶 / 秋焙茶 / 冬黑糖）
- **一杯酒從滿到空**（同個座位、同個杯、4–6 張，講「慢慢喝」）
- **客人視角看桌上**（不是俯拍、不是側拍，是「我自己在喝」的眼睛角度）
- **吧台前的硬幣 / 收銀機 / 結帳細節**（有歲月的零錢盤、手寫帳本）
- **後巷 / 倒垃圾的瞬間**（最反 IG 的 mood 反而會讓人覺得是真的店）
